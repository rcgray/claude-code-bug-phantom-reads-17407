  should acknowledge this existing mechanism and explain how trace_id/span_id   
  supplement it rather than claiming no mechanism exists.

  Finding 5: W3C Trace Context Standard Claim Not Specified                     
                                                                                
  Line 124 states "Trace context propagation follows the W3C Trace Context      
  standard" but the WPD doesn't specify which version (currently W3C Trace

  explicit about the standard version and how baggage context will be used.
